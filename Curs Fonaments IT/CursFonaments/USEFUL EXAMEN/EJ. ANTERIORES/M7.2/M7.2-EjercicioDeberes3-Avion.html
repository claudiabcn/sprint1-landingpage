Una companyia de viatges vol automatitzar el procés de validació d’edat dels passatgers abans d’embarcar. Només poden embarcar les persones que tinguin 18 anys o més. Tens disponible un array amb les edats dels diferents passatgers. Escriu un programa en JavaScript que: Recorri l’array d’edats. Per a cada passatger, comprovi si pot embarcar. Mostri per consola un missatge per cada passatger seguint aquest format: Si pot embarcar: Passatger X : embarqui. Si no pot embarcar: Passatger X : no pot embarcar. On X és el número de passatger (començant des de 1). const edades = [19, 16, 20, 95, 99, 88, 97];
